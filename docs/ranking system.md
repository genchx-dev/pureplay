Example Progression
Casual Player

A player plays:

10 matches weekly
wins 4
draws 2
loses 4

XP calculation:

(4×50)+(2×25)+(4×15)=310

Weekly XP:

310 XP

Meaning:

reaching Bronze takes a few weeks,
which feels rewarding.
Competitive Player Example

A stronger player:

Result	Count
Wins	8
Draws	1

XP:

(8×50)+(1×25)+(1×15)=440

Add:

streak bonuses,

700–1500 XP weekly.

You should NEVER let XP alone determine matchmaking.

Instead:

Matchmaking Should Use:
hidden skill rating (MMR/ELO)

progression,
